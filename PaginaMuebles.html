Página de Muebles

Proyecto académico desarrollado en Python utilizando un framework web (Django/Flask) para mostrar un catálogo de muebles. El objetivo fue crear una interfaz sencilla y funcional que permitiera visualizar productos, sus características y precios.

La aplicación incluye un diseño modular con secciones para distintos tipos de muebles, integración de imágenes representativas y enlaces a detalles de cada producto. Se trabajó en la presentación visual y en la organización clara de la información para mejorar la experiencia del usuario.

Menú principal: Vista inicial del catálogo con acceso a las diferentes categorías de muebles.

Detalle de producto: Pantalla que muestra las características, imagen y precio de cada mueble.

Carrito de compras: Funcionalidad para agregar productos seleccionados y visualizar el total antes de la compra.
