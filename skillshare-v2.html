Skillshare Versión 2.0

Descripción del Proyecto

Skillshare 2.0 es una versión mejorada respecto a la primera entrega. En lugar de desarrollarse en C#, esta versión fue construida sobre la plataforma Python utilizando Django como framework y PostgreSQL como base de datos.

El proyecto no solo muestra un mejoramiento visual de la página, sino que también incorpora animaciones, un menú principal optimizado y la integración de dos nuevas secciones: Usuario y Trabajador.

Cada sección cuenta con su propio menú y sistema de registro. Los trabajadores pueden crear servicios, mientras que los usuarios pueden visualizarlos, solicitarlos, añadir reseñas o comentarios y gestionarlos sin problema. Esta funcionalidad está pensada específicamente para los estudiantes de nuestra carrera, ofreciendo un entorno práctico y realista.

Galería

Pantalla de inicio: Aquí el usuario escoge si entra como Usuario o Trabajador.

Login: Tanto usuarios como trabajadores comparten el mismo diseño de inicio de sesión.

Menú Usuario: Muestra las categorías y servicios disponibles creados por los trabajadores.

Menú Trabajador: Permite crear nuevos servicios que luego aparecen en la vista de Usuario.
